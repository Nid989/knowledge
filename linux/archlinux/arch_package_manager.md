## Package Manager Commands

#
### Keywords
linux archlinux package manager 
#

```sh
# Sync
sudo pacman -Syy

# Update
sudo pacman -Syu

# See installed official packages
sudo pacman -Qn

# See installed unofficial packagees
sudo pacman -Qm

# See packages without versions
sudo pacman -Qq

# Remove package
sudo pacman -R <package name>
```
